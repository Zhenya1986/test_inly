## Используемый стек технологий
- FastAPI
- SQLAlchemy
- PostgreSQL(asyncpg)
- Alembic
- Docker

## Запуск:
Клонировать репозиторий на локальную машину:
```commandline
git clone 
```
Выполнить команды Docker.
```docker
docker compose up -d
```
Интерактивная документация доступна по адресу
(Реализовано через OpenAPI(Swagger)):

```
http://127.0.0.1:8000/docs#/
```
